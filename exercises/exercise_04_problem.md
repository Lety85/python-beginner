# Exercise 4: User Input

## Difficulty: Beginner

## Description
Create a program that asks the user for their name and age, then displays a personalized message.

## Learning Objectives
- Learn how to get user input with `input()`
- Convert string input to integers
- Combine multiple variables in output

## Requirements
- Ask the user for their name
- Ask the user for their age
- Calculate what year they were born (approximately)
- Print a message: `Hello [name]! You are [age] years old and were born around [year].`

## Expected Output
```
What is your name? Alice
How old are you? 25
Hello Alice! You are 25 years old and were born around 2000.
```

## Hints
- Use `input()` to get user input (it returns a string)
- Use `int()` to convert string to integer for age
- Current year is 2025
- Calculate birth year: 2025 - age
